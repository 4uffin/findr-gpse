# findr-gpse

An open source search engine experiment. Built from [Google Programmable Search Engine (GPSE)](https://programmablesearchengine.google.com/about/).

## Overview

This project is an experimental implementation of a search engine built from Google Programmable Search Engine. It serves as a learning tool to understand the core concepts and architecture of search engine systems, including web crawling, indexing, and retrieval mechanisms.
